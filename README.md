# 🛕 Tirumala Tirupati Temples Website

## 📌 Project Overview
This is a responsive temple information website that provides details about famous temples in Tirumala and Tirupati. The website helps devotees and tourists explore temple history, images, timings, and important information in a simple and user-friendly way.

## 🚀 Features
- Home page  
- Temple details page  
- Image gallery  
- Responsive design (mobile friendly)  
- Feedback section  

## 🛠 Technologies Used
- HTML  
- CSS  
- JavaScript
